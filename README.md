# GCP PySpark ETL Lab

This project demonstrates an end-to-end data engineering pipeline using GCP, PySpark, Cloud Storage, Dataproc Serverless, and BigQuery.

## Tech Stack

- Google Cloud Storage
- Dataproc Serverless / Managed Spark
- PySpark
- BigQuery
- Git / GitHub

## Project Flow

Raw banking and insurance data is uploaded to Cloud Storage, processed using PySpark on GCP, written as curated Parquet files, and loaded into BigQuery for reporting and validation.
